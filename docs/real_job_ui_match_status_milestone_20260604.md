# 実求人突合ステータスUI接続 マイルストーンログ

Date: 2026-06-04

## 実施内容

- 実求人CSV取込後の求人タブに `review_candidate` を表示するようにした。
- 取込サマリーに `接続`, `要確認候補`, `未突合`, `候補複数`, `対象外` を分けて表示した。
- 市区町村別CSV出力に `salary_review_candidate_jobs` を追加した。
- CSV取込時に東京・神奈川・埼玉・千葉の公開事業所データを読み込んでから突合するようにした。
- オフライン突合済みCSVの `match_status=review_candidate` を画面内でも維持するようにした。

## 背景

初回検証では、画面側が現在表示中の東京都データだけで突合していたため、オフライン突合では6件接続できていた求人が、画面では3件接続に落ちていた。

今回の修正で、首都圏パイロットCSV取込時の候補母集団を4都県に広げ、オフライン突合結果と画面表示が一致するようにした。

- 未突合: 3件

## 次の計画

1. 事業所詳細パネルに、接続済み求人の `match_confidence`, `match_reason`, 給与内訳をより見やすく出す。
2. `review_candidate` 行の候補上位3件を、施設名・住所・電話・スコア理由付きでレビューしやすくする。
