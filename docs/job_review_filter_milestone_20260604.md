# 求人レビュー表示フィルタ マイルストーン 2026-06-04

## 目的

実求人が増えたときにレビュー対象を探しやすくするため、求人レビュー画面に表示フィルタを追加した。

## 実装内容

- `state.jobReviewFilter` を追加。
- 求人レビュー画面に以下のフィルタを追加。
  - `レビュー対象`
  - `確認待ち`
  - `集計対象`
  - `集計対象外`
  - `全取込`
- 各フィルタボタンに件数を表示。
- 表示フィルタはレビュー表だけを切り替え、給与集計の母集団は左側の `全取込データ / 集計対象のみ` に従うことを明記。
- 媒体条件レビューなどの操作後も、現在のレビュー表示フィルタを維持。
- UI検証スクリプト `scripts/check_nurse_map_job_review_filter_ui.mjs` を追加。

## サブエージェント評価の反映

- `集計対象 / 集計対象外` は求人内容の真偽や施設評価ではなく、分析利用可否であるという前提を維持。
- `レビュー対象` と `全取込データ` が混同されないよう補足文を追加。
- 操作後もフィルタ状態が維持されることを検証に追加。
