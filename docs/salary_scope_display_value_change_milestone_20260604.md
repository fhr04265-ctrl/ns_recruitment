# 集計対象による表示値の違い マイルストーン 2026-06-04

## 目的

媒体条件・突合・給与計算条件の確認状況によって、求人給与の表示値がどう変わるかを確認できるようにした。

## 実装内容

- 求人タブに `集計対象による表示値の違い` セクションを追加。
- `全取込データ` と `集計対象のみ` を市区町村別に比較。
- 月給中央値、年俸中央値、1日あたり給与の表示値変化を表示。
- 差分表を `salary_scope_display_value_change_mvp.csv` として出力可能にした。
- `salaryCityRows()` をスコープ指定可能に拡張し、既存の集計対象切替との整合を維持。
- UI検証スクリプト `scripts/check_nurse_map_salary_scope_comparison_ui.mjs` を追加。

## サブエージェント評価の反映

- 見出しを `給与差分` ではなく `集計対象による表示値の違い` に調整。
- 表内の説明を `表示値の変化` とし、勝敗・成果予測に見える表現を避けた。
- 注記で、地域の給与水準、採用確率、応募数、入職可能性を示すものではないことを明記。
