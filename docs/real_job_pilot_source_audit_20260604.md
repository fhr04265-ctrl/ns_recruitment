# 実求人シード 出典監査メモ 2026-06-04

## 位置づけ

このCSVは、実求人パイロット投入前の最小シードデータ。網羅的な求人収集ではなく、CSV列設計、給与計算、突合、欠損レビュー、求人品質レビューCSVの動作確認を目的にする。

## 重要な制約

- 第三者求人媒体は、必ず媒体利用条件と原求人確認を行ってから本番分析に使う。
- 給与は求人票記載条件であり、実支給額ではない。
- 賞与月数だけがあり、賞与対象額が不明な場合は `annual_bonus_min/max` に入れず、警告欄へ残す。
- 上限給与が不明な場合は、下限と同額で仮置きし、`calculation_warnings` と `review_status=needs_review` で明示する。
- 求人票上の未記載は「制度なし」ではなく「求人票上確認できない」と扱う。
- 採用確率、獲得率、勝率、何％取れる、という表現は使わない。

## サブエージェントレビュー反映

- 最初の実求人パイロットは30〜50件が現実的。
- ハローワーク等を使う場合は、出典URL、取得日、求人番号、掲載/更新日、利用条件確認日を必須にする。
- 古い求人、転載、公式情報との誤認、画像・地図情報の権利帰属に注意する。
- 画面には、求人は最新とは限らない、給与は実支給額ではない、手当・賞与・固定残業代の含み方は求人票ごとに異なる、採用成功率ではない、という警告を常時表示する。
