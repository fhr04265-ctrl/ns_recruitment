# 要確認候補 比較UIマイルストーンログ

Date: 2026-06-04

## 実施内容

- 求人タブの `review_candidate` / `unmatched` 行で、候補上位3件を比較しやすくした。
- 候補ごとに、事業所名、法人名、市区町村、住所、電話、突合スコア、突合理由を表示するようにした。
- 求人票側の住所・電話・名称と候補事業所を比較し、`電話一致/差異`, `住所一致/近似/差異`, `名称一致/近似/差異` を表示するようにした。
- 未突合求人CSVに `top_candidate_details` を追加した。
- UI検証スクリプトに、候補詳細が表示されているかのチェックを追加した。
